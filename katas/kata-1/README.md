# Proyecto Universitario: Automatización en IntelliJ con Git

Este repositorio contiene un proyecto realizado como parte de un trabajo práctico para la universidad. El objetivo es desarrollar un flujo de trabajo eficiente para proyectos Java utilizando **IntelliJ IDEA** y **Git**, maximizando el uso de atajos de teclado y minimizando el uso del ratón.

## Descripción del Trabajo

El proyecto se enfoca en:
- Crear un proyecto Java en IntelliJ IDEA.
- Inicializar y gestionar un repositorio Git en el proyecto.
- Aprender y aplicar atajos de teclado esenciales para agilizar tareas como la creación de clases, navegación entre archivos, generación de código y gestión de versiones.
- Asegurar un control de versiones adecuado mediante commits regulares con mensajes claros.

## Propósito

Este trabajo está diseñado para:
1. Mejorar las habilidades prácticas con IntelliJ IDEA y Git.
2. Optimizar el tiempo y la productividad mediante el uso de herramientas avanzadas y atajos de teclado.
3. Fomentar buenas prácticas de control de versiones y organización del código.

## Uso

Este repositorio sirve como referencia para quienes estén interesados en aprender sobre:
- Automatización de flujos de trabajo en IntelliJ.
- Gestión de proyectos Java con Git.
- Uso eficiente de herramientas de desarrollo modernas.

## Información Adicional

Este proyecto fue realizado como parte de un curso universitario y puede ser útil para estudiantes, programadores principiantes y cualquier persona interesada en mejorar su flujo de trabajo al desarrollar aplicaciones en Java.
